#### `Command:`
<br/>

```bash
Router> (This is User EXEC Mode.)
```

This is Privileged EXEC Mode.
```
Router# (This is Privileged EXEC Mode.)
```

This is Configuration Mode.
```bash
Router(config)#
```

This is Interface level within configuration mode
```bash
Router(config-if)#	
```
